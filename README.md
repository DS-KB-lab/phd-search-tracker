# PhD Search Tracker

已生成的内容：

- `bookmark_sources.json`：从 Chrome 书签提取出的 `phd检索` / `北欧` / `ppd` 来源
- `opportunities.json`：当前整理出的 AI / 医疗 AI 相关机会初版
- `site/index.html`：可作为 GitHub Pages 起点的静态网页

说明：

- 当前是第一版，重点先把你已有书签和已发现的相关岗位汇总起来。
- `phd检索` 与 `北欧` 两个文件夹里当前没有直接 `.pdf` 书签。
- 若要完整覆盖“所有岗位”，还需要继续逐个高校与招聘站做第二轮抓取和清洗。
- 真正公开发布到 GitHub 前，建议先确认仓库名和是否新建公开仓库。
